About / Help / Dependencies

What this standalone mode is

A no-install BrowserQuest-inspired adventure that runs entirely in a web browser from a local file. No Node.js, no server, no CDN — just open standalone.html from the extracted repository folder.

How to play
  • Move: Arrow Keys / WASD, or click/tap a destination on the canvas.
  • Attack: Space / Enter / Attack button — sword swings in the direction of the nearest enemy.
  • Travel: Walk toward any edge arrow — transition begins automatically when you're close.
  • World Map: M key or Map button — shows all 7 areas, connections, and your current location.
  • Collect: Walk over items and relics to pick them up automatically.
  • Pause: P key or Pause button.
  • New Game: N key or New Game button.
Game systems
  • 7 Areas: Sunlit Meadow → Forest Clearing → Ancient Town (upper row); Dark Caves → Cursed Keep → Shadow Fortress → Dragon's Lair (lower row). Each area has unique biome lighting and terrain.
  • 8 Enemy types: Rat, Goblin, Bat, Skeleton, Armored Skeleton, Ogre, Evil Eye, Dragon — each with unique stats, shapes, patrol/aggro behaviour.
  • Enemy AI: Enemies patrol their spawn area. They only chase when you enter their aggro radius. Defeated enemies do not instantly respawn — they return after you revisit the area.
  • Terrain & Collision: Each area has solid obstacles (trees, rocks, water, walls, buildings). The player and enemies cannot pass through them.
  • Weapons: Iron Sword, Blue Sword, Golden Sword — boost attack power and range.
  • Armor: Cloth, Leather, Mail, Golden — each reduces incoming damage %.
  • Consumables: Health Flask (+28 HP), Fire Potion (+65 HP).
  • Relics: Ancient Relics grant score and are tracked globally.
  • XP & Levels: Defeat enemies for XP. 10 levels — each grants +4 ATK, +20 Max HP, +4 speed.
  • Sword swing: Attack triggers a visible arc animation. Damage is applied at the midpoint of the swing and only hits enemies within the arc cone.
Equipment effects
  • Iron Sword: +15 ATK, +10 range.
  • Blue Sword: +30 ATK, +15 range.
  • Golden Sword: +50 ATK, +20 range.
  • Cloth Armor: 12% damage reduction.
  • Leather Armor: 22% damage reduction.
  • Mail Armor: 35% damage reduction.
  • Golden Armor: 50% damage reduction (capped at 80%).
  • Better items automatically replace weaker ones when picked up.
Progression & saving
  • Your progress (hero stats, equipment, area, kills, relics, score) is auto-saved every 5 seconds and on page close.
  • Use Export / Import to back up or transfer save data as JSON.
  • Reset clears all progress after confirmation.
  • Save version: bq_standalone_v3
Dependencies & license
  • Runtime: Browser Canvas 2D API and localStorage. No CDN, no network.
  • Assets: All rendering is shape-based (Canvas 2D). No external images required.
  • License: BrowserQuest source is MPL 2.0; game assets are CC-BY-SA 3.0.
Keyboard shortcuts
  • Arrow Keys / WASD — Move
  • Space / Enter — Attack (sword swing)
  • M — Toggle World Map
  • P — Pause / Resume
  • N — New Game
Troubleshooting
  • Area transition: Walk toward an edge arrow — transition starts automatically when within 80px of the border (no more 5-second wait). A smooth 500ms fade animation plays.
  • Enemy respawn: Enemies do not respawn in the same session in a visited area. They return only when you re-enter after a cooldown (approx. 10 s of ticks).
  • Save not loading: If localStorage is blocked by browser policy, the game still runs but won't persist between sessions. A new v3 save key is used — old v2 saves will start fresh.
  • Duplicate loops on restart: New Game / Import both cancel the old animation frame before starting a fresh loop.
  • Equipment not applying: Only better items are auto-equipped. Check HUD — current weapon / armor are displayed.
  • Full-app setup: See README.md and standalone_info.md for Node.js server instructions.
